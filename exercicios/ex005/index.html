Principais Formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b>.

Já nesta frase, existe um termo em destaque, usando a tag <strong>. Use assim!

Itálico / ênfase

Nesta frase, temos um termo em itálico usando a tag <i>.

Já nesta frase, existe um termo em ênfase, usando a tag <em>. Use assim!

Texto Marcado

Agora olhe o texto marcado usando a tag <mark>, veja que mudarei a cor com o atributo style.

Texto grante e pequeno

Estamos criando um texto grande e um pequeno. Usando as tags, respectivamente, <big> e <small>. Lembre-se, BIG está obsoleto. Logo, faça via CSS.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usando a tag <del>

Texto inserido

Podemos maracar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. Usando a tag <ins>, observe: Não use Usando a tag <u>

Texto Sobrescrito

Para inserir coisas do tipo 2 x 23. Neste caso usa-se a tag <sup>

Texto Subscrito

Para inserir coisas do tipo H20. Neste caso, usa-se a tag <sub>
